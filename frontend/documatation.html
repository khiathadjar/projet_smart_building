Guide d’utilisation

Comment utiliser notre application web IntelliBuild

Cette page explique simplement comment naviguer dans la plateforme, se connecter, gérer les objets connectés et utiliser les espaces admin et utilisateur.

Connexion Gestion d’objets Localisation 3D
Ouvrir l’application Retour accueil

Ce que vous pouvez faire

  • Se connecter avec un compte utilisateur ou administrateur.
  • Consulter le tableau de bord et rechercher rapidement un objet.
  • Ajouter, modifier ou supprimer des objets selon le rôle.
  • Utiliser la page Localisations pour définir une position dans le bâtiment.
  • Suivre l’historique, les notifications et les paramètres.

1. Démarrage

Pour commencer, ouvrez la page d’accueil puis cliquez sur Connexion ou Inscription. Une fois connecté, l’application redirige automatiquement vers l’espace correspondant à votre rôle.

1

Créer un compte

Utilisez la page Inscription si vous n’avez pas encore de compte dans la plateforme.

2

Se connecter

Entrez votre email et votre mot de passe sur la page Connexion.

3

Accéder au dashboard

Après connexion, vous arrivez sur le dashboard admin ou utilisateur.

4

Choisir une langue

Le sélecteur de langue est disponible sur plusieurs pages pour adapter l’interface.

2. Les rôles dans l’application

Rôle Admin

Administration complète

L’administrateur peut ajouter des objets, modifier l’inventaire, consulter les localisations, suivre l’activité et gérer les paramètres avancés.

Rôle Utilisateur

Consultation et suivi

L’utilisateur peut rechercher des objets, voir ses objets, consulter les localisations, lire les notifications et gérer son profil.

3. Utilisation de l’espace administrateur

  • Dashboard: il affiche les statistiques principales, la recherche globale et les objets récents.
  • Ajouter: cette page permet de créer un nouvel objet avec son nom, son type, sa salle et sa description.
  • Objets: vous pouvez rechercher un objet, le modifier ou le supprimer.
  • Localisations: cette page montre une architecture interactive du bâtiment pour choisir une salle.
  • Notifications: vous pouvez suivre les événements importants liés à l’application.
  • Paramètres: l’administrateur y voit les utilisateurs, l’activité et l’historique admin.

4. Utilisation de l’espace utilisateur

  • Dashboard utilisateur: il affiche les objets visibles et un accès rapide aux pages utiles.
  • Mes objets: permet de retrouver les objets suivis ou utilisés par l’utilisateur.
  • Localisations utilisateur: permet de consulter ou définir un emplacement dans le bâtiment.
  • Notifications utilisateur: sert à suivre les informations importantes liées à l’activité.
  • Paramètres utilisateur: permet de gérer les informations de profil et certaines préférences.

5. Workflow conseillé

A

Connexion

Connectez-vous puis vérifiez que vous êtes sur le bon espace selon votre rôle.

B

Créer ou rechercher

Un admin peut créer un objet. Un utilisateur peut directement le rechercher.

C

Définir la salle

Utilisez la page Localisations pour sélectionner rapidement la bonne zone du bâtiment.

D

Suivre l’activité

Consultez ensuite les notifications, l’historique et les paramètres pour garder une bonne visibilité.

6. Questions fréquentes

Je ne peux pas me connecter

Vérifiez votre email, votre mot de passe et utilisez le bouton de réinitialisation si nécessaire.

Je ne vois pas les pages admin

Votre compte doit avoir le rôle administrateur. Sinon, l’application vous redirige vers l’espace utilisateur.

Comment modifier un objet

Ouvrez la page Objets, cliquez sur Modifier puis enregistrez les changements.

Comment choisir ma salle

Allez dans Localisations puis cliquez sur une salle dans la vue 3D ou utilisez la sélection rapide.
